(inner) body(bodyParam)

Assign parameter(s) from body request object.

It will pass a get body parameters which it must be defined on the request body, unless it will get a undefined result. This decorator provides the way to return body parameters in one of the next ways:
whole body object when no parameters is passing to the decorator.
partial body object when a list of parameters name is passing to the decorator.
single body parameter when pass a field name directly to the decorator.

Decorator:
Name Type Description
body ParameterDecorator Assign a single, partial or whole request body parameters.
Parameters:
Name Type Description
bodyParam string | Array.<string> | undefined

Name or parameter's list name.
Source:
  • decorators/annotations.js, line 106
Example
class GenericController {
  @Get('/') // This Response to all GET Requests for controller route.
  method(@body() req) {}
  method2(@body('username') username) {}
  method3(@body(['username', 'password']) credentials) {}
}

(inner) param(parameter)

Get a parameter value defined on the url by path.

It will return the value defined on the url path for the current method or a global middleware, this only works for single parameter, and also might be side affected if there is an any param decorator is declared for the the selected parameter it will transformed first and passed the value.

  method(@param('userId') username) {}
